I'm primarily creating this site as a reason to make a fun project I call, gnaw.js. It's a funny story, I actually picked the name before I knew what it would do. Because I think it's hilarious to have Nhawdge's GnawJS. Cause they kinda sound alike? I guess. Ahh it's okay if you're not laughing. I'm my own target audience. Anyway, my project's goal is to help developers make github.io sites. If you're not familiar with them. They are github repos that are accessed like html files. I think a lot of the world is over engineered, because engineering is fun. But also getting your name on the internet is pretty important these days, especially for web developers.

I will eventually add some sort of blog and other stuff back to this site. But we'll see how it goes. If you're looking to get in contact with me, there are some fancy links above. I hope you have a great day.
